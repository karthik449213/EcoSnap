## User Switching

To switch users anytime:

1. Tap **"Switch User"** button (top right of home screen)
2. Select a different user from DemoLoginScreen
3. Home screen updates with new user's data
4. Camera still works with new user

## Each User Can:

- ✅ View their unique streak
- ✅ View their eco-points
- ✅ Snap photos with camera
- ✅ View recycling locations on map
- ✅ Access achievements
- ✅ See profile stats
- ✅ Switch to another user
